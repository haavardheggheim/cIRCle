HotKeys and commands in cIRCle

HotKeys

  • Join a channel: CTRL+J
  • Text and color options: CTRL+E
  • Help: CTRL+H
  • About: CTRL+A
  • Quit: CTRL+Q

Commands

  • Join a channel: /j #yourChannel ( in the connectionwindow )
  • Part a channel: /part ( in the channel you want to part )
  • SetTopic: /setTopic enterYourTextHere /j #yourchannel ( in the channelwindow )
  • Chane your nick: /nick yourNewNick ( in the channelwindow )
  • List all available channels of this network: /list ( in the connectionwindow )
